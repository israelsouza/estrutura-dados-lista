## Enunciados (19 questões)

1. Implemente o algoritmo Selection Sort para ordenar uma lista de inteiros em ordem crescente. Exiba a lista antes e depois da ordenação.

--- 

2. Utilize o Selection Sort para ordenar uma lista de nomes em ordem alfabética crescente.

--- 

3. Modifique o Selection Sort para ordenar uma lista de inteiros em ordem decrescente, selecionando sempre o maior elemento.

--- 

4. Implemente o Selection Sort instrumentado para contar o número de trocas realizadas. Exiba a lista ordenada e o total de trocas.

--- 

5. Implemente o Insertion Sort para ordenar uma lista de inteiros. Exiba cada passo da inserção.

--- 

6. Demonstre a eficiência do Insertion Sort em uma lista quase ordenada, contando o número de comparações realizadas e comparando com o pior caso teórico.

--- 

7. Use o Insertion Sort para ordenar uma lista de nomes de cidades brasileiras em ordem alfabética.

--- 

8. Implemente o Bubble Sort clássico para ordenar uma lista de inteiros, exibindo cada passagem completa pelo array.

--- 

9. Implemente o Bubble Sort com a otimização da flag que interrompe o algoritmo quando nenhuma troca é realizada em uma passagem, melhorando o desempenho no melhor caso.

--- 

10. Implemente o Bubble Sort para ordenar uma lista de floats (notas de alunos) em ordem decrescente.

--- 

11. Instrumente o Bubble Sort para contar separadamente o número de comparações e trocas, e exiba um relatório de desempenho.

--- 

12. Implemente os três algoritmos e compare-os em termos de tempo de execução, número de comparações e número de trocas para diferentes tamanhos de entrada.

--- 

13. Ordene uma lista de tuplas representando produtos (nome, preço, estoque) usando Selection Sort, primeiro por preço crescente e depois por estoque decrescente.

--- 

14. Implemente o Insertion Sort em uma lista encadeada, sem usar arrays auxiliares.

--- 

15. Aplique o Bubble Sort para ordenar cada linha de uma matriz 4×4 e, em seguida, ordene a primeira coluna da matriz resultante.

--- 

16. Realize uma análise de desempenho comparativa dos três algoritmos para diferentes tamanhos de entrada (n = 10, 50, 100, 500), usando o módulo timeit para medições precisas.

--- 

17. Implemente um Selection Sort genérico que aceite uma função de comparação customizada. Aplique-o para ordenar palavras por: (a) comprimento crescente, (b) número de vogais decrescente, (c) ordem alfabética reversa.

--- 

18. Implemente os três algoritmos com visualização textual do estado do array a cada passo, destacando os elementos sendo comparados e trocados.

--- 

19. Implemente um sistema completo de ordenação de registros de alunos usando os três algoritmos. Cada aluno possui: nome, matrícula, nota final e frequência. Ordene por: 

(a) nota decrescente, 

(b) frequência decrescente com desempate por nota, 

(c) nome alfabético. 

Compare o desempenho dos três algoritmos.
